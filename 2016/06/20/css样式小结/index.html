css样式小结

发表于 2016-06-20   |   分类于 css   |     |   阅读次数

自己在开发过程中遇到的一些比较特殊的css样式，总结如下：

1. 单行文本，超过固定长度的文字用…显示

display: inline-block;    
max-width: 100px;     
white-space: nowrap;    
overflow: hidden;    
text-overflow: ellipsis;

思想：固定一行的长度，然后不换行，利用’text-overflow: ellipsis’样式将超过宽度的部分用…显示。
*font-family: Helvetica(一种广泛的西方字体);可以让…位于右下角。

2. 小三角

.drg{    
 position: relative;     
 background: rgb(123,23,123);      
}      
.drg:after{    
 position: absolute;      
 border: 20px solid transparent;    
 border-left: 20px solid rgb(23,123,123);    
 left: 100%;     
 content: '';        
}

如图：

有top、right、left、bottom四种小三角，上面的例子是left即border-left。
思想：是先把border设置成透明，然后根据需要显示对应的部分boder并用absolute进行定位。

3. 同一行图片后面的文字垂直对齐

这里主要利用了vertical-align:middel，定义行内元素的基线相对于该元素所在行的基线的垂直对齐。 

img{         
vertical-align: middle;    
}    
.font{    
display: inline-block;    
vertical-align: middle;    
}

这里的文字可以是单行，也可以是多行。

4. 如何将弹窗里面的内容居中？

样式如下： 

.overlay article {    
    position: absolute;     
    left: 50%;    
    top: 50%;    
    margin: -200px 0 0 -200px;    
    width: 400px;    
    height: 400px;    
}

5.新加一个弹窗遮罩层

相比absolute，利用fixed实现更好，可以使得遮罩层完全覆盖页面布局区域，具体样式如下： 

.overlay {      
 position: fixed;    
 left: 0;    
 right: 0;    
 top: 0;    
 bottom: 0;    
 background: rgba(0,0,0,0.8);

6.vertical-align

vertical-align:middle可以让行内元素，比如span、img垂直居中。 

<nav>    
     <img src="images/banshu.jpg">    
     <span>这是文字</span>    
     <span class="block">这是块</span>    
</nav>    

nav {                    
     height: 60px;    
     line-height: 60px;        
     background: red;        
}        
nav img {        
     width: 30px;        
     height: 30px;        
}        
nav .block {        
     display: inline-block;        
}

以上代码会使图片和文字一样垂直居中。

7.line-height

line-heigth是不包括border和padding，height是line-height,border-top(bottom),padding-top(bottom)三者之和。

8.a:nth-child()

a:nth-child(2)从第二个元素开始,不仅仅是从a元素开始算的，比如： 

<div>    
     <span>商家分类：</span>    
     <a>全部商家</a>        
     <a>品牌商家</a>        
     <a>快餐便当</a>        
     <a>特色菜系</a>        
     <a>小吃夜宵</a>        
     <a>甜品饮品</a>        
</div>

a:nth-child(2)的指的是第一个a标签”全部商家”;

9.float

float浮动，有none，left，right三个值，padding的值会对元素float的布局造成影响，比如： 

<div class="float">浮动元素</div>                
.float{                
   float:left;            
   padding: 20px;            
}

float开始的位置top：20px;left:20px;

10.max-height

max-height可以设置最大高度，有的时候需要增加overflow:auto，来增加高度到840px；

11.两列布局

第一种方法：使用float:left;这个大家基本都知道，代码如下：

.left {
    float: left;
    width: 100px;
    height: 100px;
    background: red;
}
.right{
    background: yellow;
    margin-left: 120px;
}

...

<div class="left">
    111
</div>
<div class="right">
    222
</div>

其实还有一种新的通过position: absolute;实现两列布局的方法，代码如下：

.left {
    position: absolute;
    width: 100px;
    height: 100px;
    background: red;
}
.right{
    background: yellow;
    margin-left: 120px;
}

...

<div class="left">
    111
</div>
<div class="right">
    222
</div>
