发射猫首页开发css样式问题总结

发表于 2016-07-11   |   分类于 css   |     |   阅读次数

其实很多时候只是看文档、看书、看博客，自己不用代码加以实践，时间一长就会忘掉，还是得多多复习才是。

1. 行内元素span

行内元素，比如span，是不能设置宽度和高度，对于外间距margin-top、margin-bottom也是没有效果，但是margin-left有效果。
可以添加display:inline-block(或其他)将行内元素变成块元素，那样margin-top、margin-bottom、宽度高度都会生效。

2. 给伪元素添加伪类

比如有如下伪元素： 

help::before {/* 这里是样式 */}

给上面的伪元素添加hover伪类时应该像如下写： 

help:hover::before {/* 这里是样式*/}

添加其他伪类类似。

3. 谷歌浏览器字体显示问题

在谷歌浏览器中对于那些小于12px的一律显示12px，虽然可以通过css3的一些方法显示小于12px的字体，但是还是建议字体尽量大于12px。

4. Bootstrap中的input阴影

默认情况下Bootstrap中的input输入框点击时周围会出现蓝色阴影，这是因为input增加了如下样式: 

input:focus{       
  box-shadow:  XXXX    
}

如果想要去掉蓝色阴影，直接用box-shadow: none。

5. Bootstrap中表单垂直对齐问题

有的时候在Bootstrap的form表单中我们总是垂直对不齐其中的label和input标签，可能是因为我们在form那缺少如下class样式： 

class: form-horizontal

6.百分比问题

如下代码： 

<div style="padding-left:20%">        
  <form style="width:30%"></form>        
</div>

这里form中的30%其实是剩下80%中的30%，相当于之前div大小的24%。

7. li列表除去第一个以外赋予css样式

对li列表除去第一个以外剩下的li赋予相同css样式，样式如下： 

li+li {/* 这里是样式*/}

8. border边框问题

有如下样式： 

border: 10px solid #333;        
border-style: inset(内嵌)/outset(外凸);

其实border-style就是border中的solid，如果像上面那样添加了border样式，又添加了border-style样式，那么后加的border-style会覆盖之前的border中solid属性。

9.td相对距离

table td里面如果设置width：48.6%时，本来想相对tr的48.6%，如果想设置相对当前td的48.6%，可以将td设置成块，比如display：inline-block或display：block。
