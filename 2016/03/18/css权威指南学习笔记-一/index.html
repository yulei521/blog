第一章 HTML标记与文档结构

用HTML标记某些内容的目的是为了赋予网页语义，就是给你的网页内容赋予某些用户代理(user agent)能够理解的含义。

这里用户代理主要指浏览器，屏幕阅读器，Web爬虫等；

html5相比html而言，新规定了一批结构化的标签，对相关内容进行分组，可以更好规范网页的整体结构。

第二章 CSS工作原理

2.1 剖析CSS规则

css注释使用 / 注释内容 /
有一种在样式表中链接到其他样式表的方法，应用@import 指令(是一种at规则)

@import url(css/style2.css)

要注意的是@import指令必须出现在样式表中其他样式之前，否则@import引用的样式表不会被加载。
css规则由选择符和声明两部分组成。

为文档添加样式的三种方法

  • 行内样式，写在元素标签里
  • 嵌入样式，写在style标签里
  • 链接样式，写在单独的css样式表中

优先级：行内样式>嵌入样式>链接样式

2.2 选择符

所有用于特定元素的选择符有三种：

  • 上下文选择符。基于祖先或同胞元素选择一个元素。
  • ID和类选择符。基于id和class属性的值选择元素。
  • 属性选择符。基于属性的有无和特征选择元素。

2.2.1 上下文选择符

这种“基于位置”变换某个标签样式的问题，可以用上下文选择符来解决。
上下文选择符的格式:

标签1 标签2 {声明}

其中标签2就是我们想要选择的目标，而且只有在标签1是其祖先元素(不一定是父元素)的情况下才会被选中。
上下文选择符，严格来讲叫后代组合式选择符，就是一组以空格分隔的标签名。用于选择作为指定祖先元素后代的标签。

还有一些特殊的上下文选择符，如下：

  • 子选择符 >
    标签1 > 标签2，标签2必须是标签1的子元素，标签1必须是标签2的父元素。注意：这个选择符中的标签1不能是标签2的父元素之外的其他祖先元素。

  • 紧邻同胞选择符 +
    标签1 + 标签2，标签2必须紧跟在其同胞标签1的后面。而且标签1和标签2属于同一个父元素。是标签2的样式。

  • 一般同胞选择符 ~
    标签1 ~标签2，标签2必须跟(不一定紧跟)在其同胞标签1后面。而且标签1和标签2属于同一个父元素。是标签2的样式。
  • 通用选择符 *
    表示匹配任何元素。

2.2.2 ID和类选择符

ID和类选择符为我们选择元素提供了另一套手段，利用它们可以不用考虑文档的层次结构，只要你在HTML标记中为元素添加了id和class属性，就可以在css选择符中使用ID和类名，直接选中文档中特定的区域。

  • 类属性
    就是HTML元素的class属性，body标签中包含任何HTML元素都可以添加这个属性。类选择符 .类名。选择同时存在两个类名的元素，可以这样写：

    .class1.class2 { 样式 };

  • ID属性
    ID与类的写法相似，而且表示ID选择符的#(井号)的用法，也跟表示类选择符的.(句号)类似。
    #ID值 {css声明样式}。
    另外ID可以用于页内导航，例如:
    <a href=”#ID值”></a>。
    如果链接的href属性里只有一个#，那么点击该链接会返回页面顶部，像这样<a href=”#”>回到顶部。

ID和类区别

ID的用途是在页面标记中唯一标识一个特定的元素。它能够为我们编写css规则提供必要的上下文，排除无关的标记，而只选择该上下文的标签。

类是可以应用给任意多个页面中的任意多个HTML元素的公共标识符，以便我们为这些元素应用相同的css样式。而且，使用类也让为不同标签名的元素应用相同的样式成为可能。

2.2.3 属性选择符

是基于HTML标签的属性选择元素。

  • 属性名选择符
    标签名[属性名],选择任何带有属性名的标签名。比如：
    img[title] {样式}—给带有title属性的img元素添加样式，至于title属性什么值无关紧要，只有有这个属性即可。

  • 属性值选择符
    标签名[属性名=”属性值”]，选择任何带有值为属性值的属性名的标签名。
    比如：img[title=”red flower”]{样式}—-在图片的title的值为red flower的情况下，才会为图片添加边框。

2.3 伪类

一个冒号(:)表示伪类，两个冒号(::)表示css3新增的伪元素。伪类实际上并没有类会附加到标记中的标签上，有两种：

  • UI(用户界面)伪类会在HTML元素处于某个状态时(比如鼠标指针位于链接上)，为该元素应用css样式。
  • 结构化伪类会在标记中存在某种结构上的关系时(如某个元素是一组元素中的第一个或最后一个)，为响应元素应用css样式。

2.3.1 UI伪类

UI伪类会基于特定HTML元素的状态应用样式。有：

  • 链接伪类，针对链接的伪类一共有四个。
    link—链接就在那等着用户点击
    visited—用户已经点过这个链接
    hover—鼠标指针正悬停在链接上
    active—链接正在被点击
    顺序可变，否则会得不到预期效果，可以这样记–LoVe？HA！

  • :focus伪类，表单中的文本字段在用户点击它时会获得焦点，然后用户才能在其中输入字符。例如：
    input:focus {border:input solid blue};

  • :target伪类，如果用户点击一个指向页面中其他元素的链接，则那个元素就是目标(target),可以用:target伪类选中它。
    例如:
    #more_info:target {样式}
    对于下面这个链接
    <a href=”#more_info”>More Information
    位于页面其他地方、ID为more_info的那个元素就是目标。该元素可能是这样的：
    <h2 id=”more_info”>这是：target类显示样式地址

2.3.2 结构化伪类

结构化伪类可以根据标记的结构应用样式，比如根据某些元素的父元素或前面的同胞元素是什么。主要有：

  • :first-child和：last-child
    :first-child代表一组同胞元素中的第一个元素，而：last-child代表最后一个，
    li:first-child {样式}

  • :nth-child
    e:nth-child(n)，e表示元素名，n表示一个数值(也可以使用odd代表奇数或even代表偶数)。例如：
    li:nth-child(3)–会选择一组列表中的每个第三项。

2.4 伪元素

伪元素是指文档中若有实无的元素。主要有：

  • ::first-letter伪元素
    例如： p::first-letter {font-size:300%},可以使得段首字符放大效果。

  • ::first-line伪元素
    例如： p::first-line{color:red},可以使得段的第一行字体颜色成为红色。

  • ::before和::after伪元素
    用于在特定元素前面或后面添加特殊的内容，例如:对于

    25,添加如下样式
    p.age::before {content:”Age”;}
    p.age::after {content:” years.”;}
    能得到以下结果：
    Age: 25 years.

2.5 继承

css有很多属性是可以继承的，其中相当一部分都跟文本有关，比如颜色、字体、字号。然而，也有很多css属性不能继承，因为继承这些属性没有意义，主要有元素盒子的定位和显示方式，比如边框、外边距、内边距等。

2.6 层叠

层叠就是样式表中的层叠，是一种样式在文档层次中叠加的过程。
浏览器层叠各个来源样式的顺序：

  • 浏览器默认样式
  • 用户样式表
  • 作者链接样式表(按照它们链接到页面的先后顺序)
  • 作业嵌入样式
  • 作者行内样式

层叠规则：

  • 找到应用给每个元素和属性的所有声明
  • 按照顺序和权重排序，声明也可以有权重，空格!important分号(；)用于加重声明的权重，例如：p{color:green !important; font-size:12pt;},优先级最高。
  • 按特指度排序。特指度更高的优先级高，比如p.class 优先级比p 优先级高，类名选择器比普通的标签选择符具有更高的特指度。一条规则的特指度，由它的选择符中包含多少个标签、类名和ID决定。
  • 顺序决定权重，位置靠下的(或后声明)规则胜出。
  • 设定的样式会覆盖继承的样式。

计算特指度

有一个简单的记分规则，即对每个选择符都要按下面”ICE”公式计算三个值：I-C-E
注意：0-1-12与0-2-0相比，仍然是0-2-0的特指度更高，可以看做一个三位数。
三个字母间的端横线是分隔符，并非减号，有如下记分办法:

  • 选择符中有一个ID，就在I的位置上加1;
  • 选择符中有一个类，就在c的位置上加1;
  • 选择符中有一个元素(标签)名，就在E的位置加1;
  • 得到一个三位数。此外每个选择符比前一个选择符的特指度更高，即ID>类>标签名

2.7 规则声明

css属性值主要分以下三类:

  • 文本值
    例如，font-weight:bold声明中的bold就一个文本值，文本值也叫关键字。
  • 数字值
    数字值后面都有一个单位，例如英寸或点。数字值主要分两类:绝对值和相对值。
  • 颜色值
    颜色值可以用RGB、HSL和十六进制值表示。
